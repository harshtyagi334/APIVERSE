# APIVERSE 🚀

APIVERSE is a modern and interactive API documentation platform designed to help developers explore, test, and manage APIs efficiently. Built with React, TypeScript, Vite, and Firebase, the platform provides a clean developer experience with a fast and responsive interface.

---

## 🌟 Overview

APIVERSE simplifies the process of understanding and testing APIs by providing:

- Interactive API playground
- Real-time request & response testing
- Authentication support
- Developer-friendly UI
- Fast and scalable architecture

The goal of APIVERSE is to create a seamless environment where developers can experiment with APIs without relying on external tools.

---

## ✨ Features

### 🔹 Interactive API Playground
Test API endpoints directly from the platform with real-time responses.

### 🔹 Authentication System
Secure login and authentication support using Firebase Authentication.

### 🔹 Modern UI/UX
Clean and responsive interface built for better developer productivity.

### 🔹 Fast Performance
Powered by Vite for lightning-fast development and optimized builds.

### 🔹 Scalable Architecture
Organized code structure using React and TypeScript for maintainability.

### 🔹 Firebase Integration
Backend services including authentication and hosting support.

## ⚙️ Installation & Setup

### Prerequisites

Make sure you have installed:

- Node.js
- npm or yarn

Check versions:

```bash
node -v
npm -v
```

---

## 🚀 Run Locally

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/harshtyagi334/APIVERSE.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd APIVERSE
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

The application will start on:

```bash
http://localhost:5173
```

---

## 🔐 Environment Variables

Create a `.env` file in the root directory and add your Firebase configuration:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
```
