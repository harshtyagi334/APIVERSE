# APIVerse Project Rules

- Use `lucide-react` for all icons.
- Use `framer-motion` for UI animations.
- Use `gsap` for complex sequence-based animations.
- Use `shadcn/ui` components (found in `src/components/ui`).
- Store all types in `src/types/`.
- Use `i18next` for all user-facing strings.
- Firebase is the primary backend (Firestore + Auth).
- Follow the "Recipe 1: Technical Dashboard" and "Recipe 7: Atmospheric" design guidelines for the premium futuristic feel.
